# 📊 Business Sales Performance Analytics
**Future Interns | Data Science & Analytics | Task 1**

## 🔍 Overview
Analysis of Superstore sales data to identify revenue trends, 
top-performing products, regional performance, and profit insights.

## 📁 Dataset
- **Source:** Superstore Sales Dataset (Kaggle)
- **Records:** 9,994 transactions | 21 columns
- **Period:** 2014 – 2017

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Google Colab

## 📈 Key Insights
- **Total Revenue:** $2,297,200.86
- **Profit Margin:** 12.5%
- **Top Region:** West ($700K+ in sales)
- **Top Category:** Technology (36.4% of sales)
- **#1 Product:** Canon imageCLASS 2200 Copier (~$60K)
- **Loss Alert:** Tables & Bookcases are loss-making sub-categories

## 💡 Recommendations
1. Reprice or discontinue Tables & Bookcases
2. Focus marketing on Technology in West & East regions
3. Leverage Q4 seasonal spikes for inventory planning
